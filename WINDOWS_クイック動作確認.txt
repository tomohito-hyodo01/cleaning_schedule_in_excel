【Windows環境での動作確認 - 最短手順】

============================================
  清掃スケジュール Excel変換ツール
  Windows 動作確認手順（簡易版）
============================================

■ステップ1: Pythonの確認
--------------------------------------------
1. Windowsキー + R を押す
2. 「cmd」と入力してEnter
3. 以下を入力:
   python --version

4. Python 3.9以上が表示されればOK
   表示されない場合:
   → https://www.python.org/downloads/ からインストール
   → インストール時に「Add Python to PATH」にチェック！


■ステップ2: プロジェクトフォルダに移動
--------------------------------------------
1. エクスプローラーでプロジェクトフォルダを開く
   例: C:\Projects\清掃スケジュール対応

2. アドレスバーをクリック

3. 「cmd」と入力してEnter
   → そのフォルダでコマンドプロンプトが開く


■ステップ3: セットアップ（初回のみ）
--------------------------------------------
コマンドプロンプトで以下を実行:

setup.bat

→ 自動的にライブラリがインストールされる（1-2分）


■ステップ4: アプリ起動
--------------------------------------------
run.bat

→ アプリが起動する


■ステップ5: APIキー設定（初回のみ）
--------------------------------------------
1. 「⚙️ 設定」ボタンをクリック

2. 以下のAPIキーを入力:
   your-claude-api-key-here

3. 「保存」をクリック


■ステップ6: テスト実行
--------------------------------------------
1. 「📁 画像ファイルを選択」をクリック

2. schedule_image フォルダから
   「清掃スケジュール.jpg」を選択

3. 画像のプレビューが表示されることを確認

4. 「🚀 Excel変換開始」をクリック

5. 10-30秒待つ

6. 「変換完了」ダイアログが表示される

7. デスクトップに Excel ファイルが作成される


■動作確認チェックリスト
--------------------------------------------
□ アプリが起動する
□ 設定でAPIキーが保存できる
□ 画像が選択できる
□ 画像のプレビューが表示される
□ 変換ボタンが有効になる
□ プログレスバーが動く
□ Excel ファイルが生成される
□ デスクトップに保存される
□ Excel ファイルが開ける


■実行ファイル版のビルド（配布用）
--------------------------------------------
動作確認が完了したら:

build.bat

→ dist\CleaningSchedule.exe が生成される
→ このファイルを配布すればPython不要で動作


■トラブルシューティング
--------------------------------------------
【エラー: python が認識されない】
→ Python をインストール（Add to PATH にチェック）
→ コマンドプロンプトを再起動

【エラー: モジュールがない】
→ setup.bat を再実行

【エラー: APIキーが無効】
→ APIキーを再入力（コピペ推奨）

【エラー: アプリが起動しない】
→ pip install -r requirements.txt を実行


■詳しい説明
--------------------------------------------
WINDOWS_テスト手順.md を参照してください
